## 문제 링크<br>
 [https://www.acmicpc.net/problem/9461](https://www.acmicpc.net/problem/9461)<br>

#### 전체코드<br>
```cpp
#include<iostream>

#define SIZE 100

using namespace std;

int main(void){
    long long dp[SIZE];
    int T,n;

    cin>>T;

    for(int t=0; t<T; ++t) {
        cin >> n;

        dp[0] = 1;
        dp[1] = 1;
        dp[2] = 1;
        dp[3] = 2;
        dp[4] = 2;

        for (int i = 5; i < n; ++i) {
            dp[i] = dp[i - 1] + dp[i - 5];
        }
        cout << dp[n - 1] << endl;
    }

    return 0;
}
```
