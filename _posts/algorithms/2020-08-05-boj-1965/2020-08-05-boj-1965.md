## 문제 링크<br>
 [https://www.acmicpc.net/problem/1965](https://www.acmicpc.net/problem/1965)<br>


#### 전체코드<br>

```cpp
#include<iostream>
#define SIZE 1001

using namespace std;

int main(){
    int arr[2][SIZE]={0};
    int dp[SIZE]={0};
    int n;
    cin>>n;
    int max = 0;
    for(int i=0; i<n; i++){
        cin>>arr[0][i];
        if(arr[0][i]>max)
            max = arr[0][i];
    }
    dp[1] = arr[0][0];
    arr[1][0] = 1;
    for(int i=2; i<=n; i++){
        dp[i] = max;
    }
    for(int i=1; i<n; i++){
        int checkNum = arr[0][i];
        int checkIndex=0;
        for(int index=1; index<n; index++){
            if(checkNum>dp[index]&&checkNum<=dp[index+1]){
                checkIndex = index;
                break;
            }
        }
        dp[checkIndex+1] = checkNum;
        arr[1][i] = checkIndex+1;
    }

    max = 0;
    for(int i=0; i<n; i++){
        if(arr[1][i]>max)
            max = arr[1][i];
    }
    cout<<max<<endl;


    return 0;
}
```

## 풀이<br>
=== 방법 ===  
1.입력을 전부 받기  
2.각 상자가 오름차순에서 몇 번째에 해당되는지 확인  
3.가장 긴 오름차순 번호를 추출

=== 사용된 변수 ===  

| 변수        | 설명           |
| ------------- | ------------- |
| arr[2][1001] | arr[0]은 상자의 크기를, arr[1]은 해당 상자의 오름차순 해당 번호를 나타냄.|  
| dp[1001] | 오름차순 번호대로 해당되는 B 전깃줄 위치 중 작은 것 저장.|
| max | 입력 받을 때 최대 상자 크기를 기억. dp[1001]를 max로 초기화 시킴.<br> dp 걔산 후 최종 가장 긴 오름차순 번호를 찾을 때 재사용됨. |
