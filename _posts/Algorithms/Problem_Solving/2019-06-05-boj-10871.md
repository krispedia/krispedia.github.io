## 문제 링크<br>
 [https://www.acmicpc.net/problem/10871](https://www.acmicpc.net/problem/10871)<br>

#### 전체코드<br>
```cpp
#include<iostream>

using namespace std;

int main(void){
    ios_base::sync_with_stdio(false);
    int N, X;
    cin>>N>>X;

    for(int i=0; i<N; i++){
        int num;
        cin>>num;
        if(num<X)
            cout<<num<<" ";
    }

    return 0;
}
```
