## 문제 링크<br>
 [https://www.acmicpc.net/problem/1010](https://www.acmicpc.net/problem/1010)<br>

#### 전체코드<br>
```cpp
#include<iostream>

#define SIZE 30

using namespace std;

int dp[SIZE][SIZE] = {0};

int sol(int i, int j){
    if(dp[i][j]!=0)
        return dp[i][j];
    else if(i==j){
        dp[i][j] = 1;
        return dp[i][j];
    }
    else if(i==1){
        dp[i][j] = j;
        return dp[i][j];
    }
    else{
        int sum = 0;
        for(int k=j; k>=i; --k)
            sum += sol(i-1, k-1);
        dp[i][j] = sum;
        return dp[i][j];
    }
}
int main(void){
    int T;
    int N, M;

    cin>>T;
    for(int t=0; t<T; ++t){
        cin>>N>>M;
        cout<<sol(N,M)<<endl;
    }
    return 0;
}
```

## 풀이<br>
간단한 예시로 생각해본다면  

ex) N=3, M=4 인경우  

|-|-|-|
|(2,4)|(2,3)|(2,2)|   
|(1,3)|(1,2)|(1,1)|  

이 경우 3+2+1=6 으로 최종 결과가 6개이다.  

N의 위에서부터 하나씩 선택한 후에 최종 N이 1개가 남은 경우 가능한 개수를 세는 방식을 택했다.
